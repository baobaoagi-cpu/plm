# Three-Loop Durable Memory

This directory is the repository-backed source of truth for ongoing Codex work.

Read in this order before each iteration:

1. `../minimax-duplex-voice-spec-v1.0.md`
2. `current-plan.md`
3. `progress.json`
4. `blockers.md`
5. the latest file under `runs/`
6. relevant files under `evidence/` and `decisions/`

Evidence labels:

- `VERIFIED`: supported by tests, a real provider run, reproducible measurements, or explicit
  official documentation.
- `INFERRED`: reasoned from evidence but not directly exercised.
- `UNKNOWN`: insufficient evidence; requires a spike, contract test, official source, or human input.

Only the Governance Loop may advance the milestone. Build output alone cannot mark a milestone
complete.
